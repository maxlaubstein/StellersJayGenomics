This repository contains the scripts and files used to infer effective migration surfaces from Steller's Jay genomic data across the western United States and Canada.

There are 3 separate params files to be used to run EEMS from three different starting seeds. I ran each in a different screen session to basically run them in parallel. For example, in the screen for seed 1 I ran:

```../eems/runeems_snps/src/runeems_snps --params params_350demes_1```
